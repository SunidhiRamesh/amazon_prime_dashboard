# 📊 Amazon Prime Video Dashboard

An interactive **Power BI dashboard** built to analyze Amazon Prime Video's content library. The dashboard provides insights into movies, TV shows, genres, ratings, release trends, and overall platform statistics through interactive visualizations.

---

## 🚀 Features

- 📈 Interactive Power BI dashboard
- 🎬 Total Movies & TV Shows analysis
- ⭐ Content rating distribution
- 🎭 Genre-wise content analysis
- 🎥 Director-wise statistics
- 📅 Release year trend analysis
- 📊 Movies vs TV Shows comparison
- 🔍 Easy-to-understand visual insights

- Distribution of Movies vs TV Shows
- Top Genres

## 📌 Business Insights

This dashboard helps answer questions such as:

- Which genres are most popular on Amazon Prime Video?
- How has the content library grown over the years?
- What percentage of the catalog consists of Movies versus TV Shows?
- Which content ratings appear most frequently?
- How diverse is the platform in terms of genres and directors?
